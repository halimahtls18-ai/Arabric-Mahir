# 🌟 UAS Chatbot PBA - ARABIC MAHIR 🌟

Aplikasi Chatbot pintar berbasis Artificial Intelligence (AI) yang dirancang khusus sebagai media pembelajaran mandiri Bahasa Arab untuk tingkat Pemula. Aplikasi ini berfokus pada peningkatan keterampilan **Maharah Kalam (Berbicara)** dan pemahaman **Sharaf (Perubahan Kata/Tasrif)** dengan pendekatan yang interaktif dan menyenangkan.

## 🎯 Fitur & Mode Pembelajaran
Aplikasi ini dilengkapi dengan antarmuka grafis terminal yang menarik (menggunakan pustaka `Rich`) dan menyediakan 3 fitur utama:
1. **🗣️ Mode Percakapan Santai:** Ruang bagi mahasiswa untuk melatih percakapan sehari-hari menggunakan bahasa Arab secara aktif bersama AI.
2. **🧠 Mode Games Asah Otak:** Fitur tebak-tebakan interaktif mengenai perubahan kata (*tasrif*) untuk menguji kekuatan hafalan Sharaf mahasiswa.
3. **🎭 Mode Tantangan Berbicara:** Fitur di mana AI memberikan instruksi atau tema khusus untuk memicu mahasiswa menyusun kalimat bahasa Arab secara mandiri.

## 🛠️ Teknologi yang Digunakan
* **Bahasa Pemrograman:** Python 3
* **AI Engine:** Google Gemini API (`gemini-2.5-flash`) via `google-genai` SDK
* **Antarmuka (UI):** `Rich` Library (untuk tampilan panel warna magenta dan teks interaktif)
* **Manajemen Environment:** `python-dotenv` (mengamankan API Key agar tidak bocor)
